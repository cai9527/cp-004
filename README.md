## 🎯 核心功能模块
### 1. 巡检任务管理
- 任务列表 ：按状态筛选（待执行/进行中/已完成），支持搜索
- 任务详情 ：任务信息、检查项清单、执行记录
- 现场检查记录 ：逐项检查、拍照取证、GPS定位、离线暂存、草稿自动保存
### 2. 安全隐患管理
- 隐患列表 ：多维度筛选（等级/状态/分类/时间）
- 隐患上报 ：项目选择、等级分类、标题描述、GPS定位、拍照取证（最多9张）、整改建议
- 隐患详情 ：完整信息展示、操作时间轴
### 3. 整改跟踪闭环
- 整改列表 ：角色视角切换（我负责的/我指派的/我复查的）、状态筛选、逾期红色高亮
- 整改详情 ：
  - 隐患信息完整展示
  - 基于角色和状态的动态操作按钮
  - 操作时间轴：上报 → 指派 → 整改 → 复查 → 闭环
  - 三种操作弹窗：指派整改、提交整改、复查不通过原因
### 4. 数据统计分析
- 工作台 ：今日待办、本月统计、快捷入口
- 数据分析 ：隐患趋势图、等级分布、分类统计、整改率
### 5. 离线同步机制
- 网络状态切换 ：在线/离线模式手动切换
- 离线数据采集 ：离线时数据自动暂存本地
- 同步队列管理 ：待同步/已同步/同步失败分类统计
- 自动同步 ：网络恢复后自动同步
- 手动同步 ：一键同步全部、单条同步、删除失败记录
- 重试机制 ：最多自动重试5次
### 6. 个人中心
- 用户信息卡片（头像、姓名、角色、部门、工号）
- 统计数据（本月任务/发现隐患/整改完成）
- 常用功能菜单
- 系统设置（网络状态切换、退出登录）
## 🔧 技术架构
技术栈 说明 Uniapp 跨平台框架，支持H5、微信小程序、App Vue 2.x 前端框架，组件化开发 Vuex 全局状态管理（用户、任务、隐患、同步队列） uView UI UI组件库 SCSS 样式预处理 Mock数据 完整模拟数据，支持独立运行
